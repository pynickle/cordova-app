## 我的Cordova app

这个应用是我用来学习cordova的（现在如此），主要涉及众多插件，现已涉及：

- cordova-plugin-file : **文件操作**
- cordova-plugin-camera : **相机操作**
- cordova-plugin-file-transfer : **文件操作**

## 文件结构解释

- www   *主文件夹*
  - index   *首页*
  - logo.png   *图标*
  - src   *功能文件夹*
    - camera   *相机*
      - index   *相机功能页*
    - file   *文件*
      - index   *文件功能页*
    - storage   *存储*
      - index   *存储功能页*
